# Acid Graphics

Acid Graphics draws inspiration from 90s rave culture and acid rock. It's a rebellious design language that emphasizes visual impact and emotional expression.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-cream` | `#f0f0e0` | Main background |
| `--acid-green` | `#ccff00` | Primary accent |
| `--acid-pink` | `#ff00ff` | Secondary accent |
| `--acid-cyan` | `#00ffff` | Decorative |
| `--ink-black` | `#1a1a1a` | Borders/Text |

### Effects
*   **Hard Shadow**: `box-shadow: 8px 8px 0 #1a1a1a`
*   **Text Shadow**: Multi-layer offset shadows
*   **Skew**: `skewX(-5deg)` for dynamic feel
*   **Thick Border**: `3px solid #1a1a1a`

## Usage Example
