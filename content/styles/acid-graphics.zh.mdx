# 酸性设计 (Acid Graphics)

酸性设计风格从90年代的锐舞文化和酸性摇滚中汲取灵感。它是一种反叛传统的设计语言，强调视觉冲击力和情感表达。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-cream` | `#f0f0e0` | 主背景 |
| `--acid-green` | `#ccff00` | 主要强调色 |
| `--acid-pink` | `#ff00ff` | 次要强调色 |
| `--acid-cyan` | `#00ffff` | 装饰色 |
| `--ink-black` | `#1a1a1a` | 边框/文字 |

### 效果
*   **硬阴影**: `box-shadow: 8px 8px 0 #1a1a1a`
*   **文本阴影**: 多层偏移阴影
*   **倾斜**: `skewX(-5deg)` 创造动感
*   **粗边框**: `3px solid #1a1a1a`

.acid-card {
  background: #ffffff;
