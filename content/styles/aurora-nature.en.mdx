# Aurora Nature

Aurora Nature draws inspiration from the aurora borealis, forests, and natural landscapes. It creates a tranquil and sustainable visual experience.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-forest` | `#0d1f1a` | Main background |
| `--aurora-green` | `#4ade80` | Primary accent |
| `--aurora-mint` | `#22d3a0` | Secondary accent |
| `--aurora-teal` | `#2dd4bf` | Decorative |
| `--text-mist` | `#e8f5e9` | Body text |

### Effects
*   **Aurora Glow**: Multiple `radial-gradient` layers
*   **Soft Glow**: `box-shadow` with low opacity
*   **Organic Shapes**: Large border radius (`border-radius: 16px`)

## Usage Example

```css
.aurora-card {
  background: linear-gradient(135deg, rgba(74, 222, 128, 0.08), rgba(26, 46, 37, 0.9));
  border: 1px solid rgba(74, 222, 128, 0.2);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3), 0 0 20px rgba(74, 222, 128, 0.1);
  color: #e8f5e9;
  border-radius: 16px;
  padding: 20px;
}
```
