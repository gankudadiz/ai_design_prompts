# 极光自然 (Aurora Nature)

极光自然风格从北极光、森林和自然景观中汲取灵感。它创造了一种宁静、可持续的视觉体验。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-forest` | `#0d1f1a` | 主背景 |
| `--aurora-green` | `#4ade80` | 主要强调色 |
| `--aurora-mint` | `#22d3a0` | 次要强调色 |
| `--aurora-teal` | `#2dd4bf` | 装饰色 |
| `--text-mist` | `#e8f5e9` | 正文文本 |

### 效果
*   **极光光晕**: 多层 `radial-gradient` 叠加
*   **柔和发光**: `box-shadow` 配合低透明度
*   **有机形状**: 大圆角 (`border-radius: 16px`)

## 使用示例

```css
.aurora-card {
  background: linear-gradient(135deg, rgba(74, 222, 128, 0.08), rgba(26, 46, 37, 0.9));
  border: 1px solid rgba(74, 222, 128, 0.2);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3), 0 0 20px rgba(74, 222, 128, 0.1);
  color: #e8f5e9;
  border-radius: 16px;
  padding: 20px;
}
```
