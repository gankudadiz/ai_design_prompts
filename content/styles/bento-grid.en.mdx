# Bento Grid

Bento Grid is a layout trend popularized by Apple (promotional materials) and Microsoft (Metro UI). It organizes content into distinct, rectangular compartments, much like a Japanese bento box.

## Design Tokens

### Layout
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--grid-gap` | `24px` | Space between cards |
| `--card-radius` | `32px` | Card corner radius |
| `--card-bg` | `#ffffff` | Card background |

### Principles
*   **Modularity**: Everything is a box. Boxes can span multiple columns or rows.
*   **Hierarchy**: Size indicates importance. Larger boxes are for hero content.
*   **Visual Interest**: Mix text-heavy cards with image-heavy cards.

## Usage Example

```css
.bento-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: 300px;
  gap: 24px;
}

.bento-card {
  background: white;
  border-radius: 32px;
  padding: 32px;
  overflow: hidden;
}

.bento-card.wide {
  grid-column: span 2;
}

.bento-card.tall {
  grid-row: span 2;
}
```
