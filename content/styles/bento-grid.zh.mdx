# Bento Grid (便当盒网格)

Bento Grid 是一种由 Apple（宣传材料）和 Microsoft (Metro UI) 推广的布局趋势。它将内容组织成独特的矩形隔间，就像日本的便当盒一样。

## 设计变量

### 布局
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--grid-gap` | `24px` | 卡片间距 |
| `--card-radius` | `32px` | 卡片圆角半径 |
| `--card-bg` | `#ffffff` | 卡片背景 |

### 原则
*   **模块化**: 一切都是盒子。盒子可以跨越多列或多行。
*   **层级**: 大小表示重要性。较大的盒子用于主要内容。
*   **视觉趣味**: 混合使用文字密集的卡片和图片密集的卡片。

## 使用示例

```css
.bento-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: 300px;
  gap: 24px;
}

.bento-card {
  background: white;
  border-radius: 32px;
  padding: 32px;
  overflow: hidden;
}

.bento-card.wide {
  grid-column: span 2;
}

.bento-card.tall {
  grid-row: span 2;
}
```
