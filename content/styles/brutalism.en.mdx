# Brutalism

Neo-Brutalism is a contemporary take on the architectural style of Brutalism. In web design, it rejects standard UI patterns in favor of raw, bold, and sometimes "ugly" aesthetics.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-color` | `#ffde00` | Bright background |
| `--border-color` | `#000000` | Thick borders |
| `--accent-color` | `#ff00ff` | Buttons/Highlights |
| `--text-color` | `#000000` | High contrast text |

### Effects
*   **Border**: `3px solid black`
*   **Box Shadow**: `5px 5px 0px 0px black` (Hard shadow)
*   **Border Radius**: `0px` (Sharp corners)

## Usage Example

```css
.brutalist-btn {
  background-color: #ff00ff;
  border: 3px solid black;
  box-shadow: 5px 5px 0px 0px black;
  color: black;
  font-weight: bold;
  text-transform: uppercase;
  transition: all 0.1s;
}

.brutalist-btn:active {
  transform: translate(2px, 2px);
  box-shadow: 3px 3px 0px 0px black;
}
```
