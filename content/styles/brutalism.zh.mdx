# 野兽派 (Brutalism)

新野兽派是对野兽派建筑风格的一种当代诠释。在网页设计中，它拒绝标准的 UI 模式，转而采用原始、大胆，有时甚至是“丑陋”的美学。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-color` | `#ffde00` | 明亮的背景 |
| `--border-color` | `#000000` | 粗边框 |
| `--accent-color` | `#ff00ff` | 按钮/高光 |
| `--text-color` | `#000000` | 高对比度文本 |

### 效果
*   **边框**: `3px solid black`
*   **盒阴影**: `5px 5px 0px 0px black` (硬阴影)
*   **圆角半径**: `0px` (尖角)

## 使用示例

```css
.brutalist-btn {
  background-color: #ff00ff;
  border: 3px solid black;
  box-shadow: 5px 5px 0px 0px black;
  color: black;
  font-weight: bold;
  text-transform: uppercase;
  transition: all 0.1s;
}

.brutalist-btn:active {
  transform: translate(2px, 2px);
  box-shadow: 3px 3px 0px 0px black;
}
```
