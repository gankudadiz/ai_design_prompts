# Corporate Blue

Corporate Blue is a design style that embodies professionalism, trustworthiness, and reliability. It uses a classic blue and white color palette that has been proven effective in enterprise and business applications for decades.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-color` | `#f0f4f8` | Main Background |
| `--primary` | `#1a365d` | Navy headings, key text |
| `--accent` | `#0056b3` | Action buttons, links |
| `--surface` | `#ffffff` | Cards, panels, modals |
| `--border` | `#cbd5e0` | Borders, dividers |
| `--text-body` | `#2d3748` | Body text |
| `--text-muted` | `#718096` | Secondary text |

### Principles
*   **Trust Through Color**: Blue is associated with stability, security, and professionalism.
*   **Clear Hierarchy**: Use size and color weight to establish visual order.
*   **Minimal Distraction**: Avoid decorative elements that don't serve a purpose.
*   **Consistent Spacing**: Use consistent padding and margins throughout.

## Usage Example

```css
/* Corporate Button */
.corporate-btn {
```

## Best Practices

1.  **Use Blue Strategically**: Reserve the primary blue for important actions and key information.
2.  **Maintain White Space**: Don't crowd elements; give content room to breathe.
3.  **Keep it Simple**: Avoid gradients, complex shadows, or decorative elements.
4.  **Focus on Content**: The design should support the content, not compete with it.
