# 企业孟菲斯 (Corporate Memphis)

企业孟菲斯（也称为 Alegria）是一种在大型科技公司插图中广泛使用的扁平几何艺术风格。它的设计旨在看起来友好、包容且可扩展。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-color` | `#f2f6ff` | 浅色背景 |
| `--primary` | `#5850ec` | 企业蓝 |
| `--secondary` | `#ffadad` | 柔和粉 |
| `--text-color` | `#2d3748` | 深灰色文本 |

### 原则
*   **扁平**: 插图中没有渐变或纹理。
*   **几何**: 使用简单的形状（圆形、正方形）来构建复杂的场景。
*   **柔和**: 圆角 (`border-radius: 16px+`) 和柔和的阴影。

## 使用示例

```css
.corp-card {
  background: white;
  border-radius: 24px;
  box-shadow: 0 20px 25px -5px rgba(0, 0, 0, 0.1), 
              0 10px 10px -5px rgba(0, 0, 0, 0.04);
  padding: 32px;
  text-align: center;
}

.corp-btn {
  background-color: #5850ec;
  border-radius: 50px; /* 药丸形状 */
  color: white;
  padding: 12px 32px;
  font-weight: 600;
  border: none;
}
```
