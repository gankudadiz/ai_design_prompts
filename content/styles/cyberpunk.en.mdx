# Cyberpunk

Cyberpunk style draws inspiration from sci-fi and dystopian futures. It is characterized by dark backgrounds, bright neon colors, and glitch effects.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-dark` | `#050505` | Main background |
| `--neon-cyan` | `#00ffea` | Primary accent |
| `--neon-pink` | `#ff00ff` | Secondary accent |
| `--neon-yellow` | `#fcee0a` | Warning/Highlight |

### Effects
*   **Glow**: `box-shadow: 0 0 10px #00ffea, 0 0 20px #00ffea`
*   **Clip Path**: `polygon(0 0, 100% 0, 100% 85%, 95% 100%, 0 100%)` (Chamfered corner)
*   **Glitch**: Animation mimicking signal distortion.

## Usage Example

```css
.cyber-card {
  background: rgba(0, 0, 0, 0.8);
  border: 1px solid #00ffea;
  box-shadow: 0 0 10px rgba(0, 255, 234, 0.5);
  color: #00ffea;
  clip-path: polygon(0 0, 100% 0, 100% 90%, 90% 100%, 0 100%);
  padding: 20px;
}
```
