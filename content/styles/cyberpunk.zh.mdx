# 赛博朋克 (Cyberpunk)

赛博朋克风格从科幻和反乌托邦未来中汲取灵感。它的特点是深色背景、明亮的霓虹色和故障效果。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-dark` | `#050505` | 主要背景 |
| `--neon-cyan` | `#00ffea` | 主要强调色 |
| `--neon-pink` | `#ff00ff` | 次要强调色 |
| `--neon-yellow` | `#fcee0a` | 警告/高光 |

### 效果
*   **发光**: `box-shadow: 0 0 10px #00ffea, 0 0 20px #00ffea`
*   **剪切路径**: `polygon(0 0, 100% 0, 100% 85%, 95% 100%, 0 100%)` (切角)
*   **故障**: 模仿信号失真的动画。

## 使用示例

```css
.cyber-card {
  background: rgba(0, 0, 0, 0.8);
  border: 1px solid #00ffea;
  box-shadow: 0 0 10px rgba(0, 255, 234, 0.5);
  color: #00ffea;
  clip-path: polygon(0 0, 100% 0, 100% 90%, 90% 100%, 0 100%);
  padding: 20px;
}
```
