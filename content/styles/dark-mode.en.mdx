# Dark Mode

Dark Mode is a color scheme that uses light-colored text, icons, and graphical user interface elements on a dark background.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-root` | `#121212` | Main background |
| `--bg-surface` | `#1e1e1e` | Card/Modal background |
| `--text-primary` | `#e0e0e0` | Main text (87% opacity) |
| `--text-secondary` | `#b0b0b0` | Secondary text (60% opacity) |
| `--accent` | `#bb86fc` | Primary action |

### Principles
*   **Elevation**: In dark mode, elevation is often expressed by lighter surface colors rather than shadows.
*   **Contrast**: Ensure text has a contrast ratio of at least 15.8:1 against the background.

## Usage Example

```css
body {
  background-color: #121212;
  color: #e0e0e0;
}

.card {
  background-color: #1e1e1e;
  border: 1px solid #333333;
  border-radius: 8px;
  padding: 16px;
}
```
