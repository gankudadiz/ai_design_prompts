# 深色模式 (Dark Mode)

深色模式是一种配色方案，它在深色背景上使用浅色文本、图标和图形用户界面元素。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-root` | `#121212` | 主要背景 |
| `--bg-surface` | `#1e1e1e` | 卡片/模态框背景 |
| `--text-primary` | `#e0e0e0` | 主要文本 (87% 不透明度) |
| `--text-secondary` | `#b0b0b0` | 次要文本 (60% 不透明度) |
| `--accent` | `#bb86fc` | 主要操作 |

### 原则
*   **层级**: 在深色模式下，层级通常通过较浅的表面颜色而不是阴影来表达。
*   **对比度**: 确保文本与背景的对比度至少为 15.8:1。

## 使用示例

```css
body {
  background-color: #121212;
  color: #e0e0e0;
}

.card {
  background-color: #1e1e1e;
  border: 1px solid #333333;
  border-radius: 8px;
  padding: 16px;
}
```
