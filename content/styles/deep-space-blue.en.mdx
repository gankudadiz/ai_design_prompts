# Deep Space Blue

This style is perfect for developer tools, dashboards, and documentation sites that want to convey a sense of technical precision and depth.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-primary` | `#0a0e1a` | Main background |
| `--bg-secondary` | `#111520` | Card/Section background |
| `--text-primary` | `#e0e0e0` | Body text |
| `--accent-primary` | `#00d4ff` | Links, Buttons, Highlights |
| `--accent-secondary` | `#00ffcc` | Prompts, Success states |
| `--border-color` | `#333333` | Dividers, Borders |

### Typography
*   **Font Family**: `'Courier New', monospace`
*   **Scale**: 14px (Body), 16px (Nav), 32px (Hero)

## Usage Example

```css
.btn {
  background: transparent;
  border: 1px solid #00d4ff;
  color: #00d4ff;
  font-family: 'Courier New', monospace;
  padding: 12px 24px;
  transition: all 0.2s;
}

.btn:hover {
  background: #00d4ff;
  color: #0a0e1a;
}
```
