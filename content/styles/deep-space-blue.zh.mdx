# Deep Space Blue (深空科幻)

这种风格非常适合开发者工具、仪表盘和文档站点，旨在传达技术精确度和深度感。

## 设计变量 (Design Tokens)

### 颜色
| Token | 值 | 用途 |
| :--- | :--- | :--- |
| `--bg-primary` | `#0a0e1a` | 主背景 |
| `--bg-secondary` | `#111520` | 卡片/区块背景 |
| `--text-primary` | `#e0e0e0` | 正文文本 |
| `--accent-primary` | `#00d4ff` | 链接, 按钮, 高亮 |
| `--accent-secondary` | `#00ffcc` | 提示符, 成功状态 |
| `--border-color` | `#333333` | 分割线, 边框 |

### 排版
*   **字体族**: `'Courier New', monospace`
*   **字号**: 14px (正文), 16px (导航), 32px (大标题)

## 使用示例

```css
.btn {
  background: transparent;
  border: 1px solid #00d4ff;
  color: #00d4ff;
  font-family: 'Courier New', monospace;
  padding: 12px 24px;
  transition: all 0.2s;
}

.btn:hover {
  background: #00d4ff;
  color: #0a0e1a;
}
```
