# Flat Design

Flat design is a style of interface design emphasizing minimum use of stylistic elements that give the illusion of three dimensions (such as the use of drop shadows, gradients or textures) and is focused on a minimalist use of simple elements, typography and flat colors.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-color` | `#ecf0f1` | Background |
| `--primary` | `#3498db` | Action Blue |
| `--danger` | `#e74c3c` | Alert Red |
| `--text` | `#2c3e50` | Main Text |

### Principles
*   **No Depth**: Eliminate shadows, bevels, and gradients.
*   **Typography**: Use sans-serif fonts to promote readability.
*   **Color**: Use a distinct color palette to guide the user.

## Usage Example

```css
.flat-btn {
  background-color: #e74c3c;
  color: white;
  border: none;
  border-radius: 4px;
  padding: 10px 20px;
  /* No box-shadow */
}

.flat-btn:hover {
  background-color: #c0392b; /* Slightly darker shade for hover */
}
```
