# 扁平化设计 (Flat Design)

扁平化设计是一种界面设计风格，强调尽量少地使用产生三维错觉的风格元素（如阴影、渐变或纹理），专注于极简主义地使用简单元素、排版和扁平颜色。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-color` | `#ecf0f1` | 背景 |
| `--primary` | `#3498db` | 动作蓝 |
| `--danger` | `#e74c3c` | 警告红 |
| `--text` | `#2c3e50` | 主要文本 |

### 原则
*   **无深度**: 消除阴影、斜面和渐变。
*   **排版**: 使用无衬线字体以提高可读性。
*   **颜色**: 使用独特的调色板来引导用户。

## 使用示例

```css
.flat-btn {
  background-color: #e74c3c;
  color: white;
  border: none;
  border-radius: 4px;
  padding: 10px 20px;
  /* 无 box-shadow */
}

.flat-btn:hover {
  background-color: #c0392b; /* 悬停时稍暗的阴影 */
}
```
