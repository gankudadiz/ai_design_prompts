# Glassmorphism

Glassmorphism is a popular design trend that uses the properties of glass (transparency and blur) to create a sense of hierarchy and depth.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--glass-bg` | `rgba(255, 255, 255, 0.25)` | Card background |
| `--glass-border` | `rgba(255, 255, 255, 0.18)` | Border color |
| `--glass-shadow` | `0 8px 32px 0 rgba(31, 38, 135, 0.37)` | Depth shadow |
| `--text-color` | `#ffffff` | Primary text (on dark gradients) |

### Effects
*   **Backdrop Filter**: `blur(10px)`
*   **Border Radius**: `16px` (Rounded corners enhance the glass effect)

## Usage Example

```css
.glass-card {
  background: rgba(255, 255, 255, 0.25);
  box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.37);
  backdrop-filter: blur(4px);
  -webkit-backdrop-filter: blur(4px);
  border-radius: 10px;
  border: 1px solid rgba(255, 255, 255, 0.18);
}
```
