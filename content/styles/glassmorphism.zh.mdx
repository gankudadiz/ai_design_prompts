# 玻璃拟态 (Glassmorphism)

玻璃拟态是一种流行的设计趋势，利用玻璃的特性（透明度和模糊）来创造层级感和深度。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--glass-bg` | `rgba(255, 255, 255, 0.25)` | 卡片背景 |
| `--glass-border` | `rgba(255, 255, 255, 0.18)` | 边框颜色 |
| `--glass-shadow` | `0 8px 32px 0 rgba(31, 38, 135, 0.37)` | 深度阴影 |
| `--text-color` | `#ffffff` | 主要文本（在深色渐变上） |

### 效果
*   **背景模糊**: `blur(10px)`
*   **圆角半径**: `16px` (圆角增强玻璃效果)

## 使用示例

```css
.glass-card {
  background: rgba(255, 255, 255, 0.25);
  box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.37);
  backdrop-filter: blur(4px);
  -webkit-backdrop-filter: blur(4px);
  border-radius: 10px;
  border: 1px solid rgba(255, 255, 255, 0.18);
}
```
