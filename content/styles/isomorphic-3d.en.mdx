# Isomorphic 3D

Isomorphic 3D (often just called 3D illustration) brings depth and dimension to web design. It often uses an isometric perspective where the x, y, and z axes are 120 degrees apart.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-color` | `#f0f2f5` | Soft background |
| `--primary` | `#6c63ff` | Vibrant accent |
| `--shadow` | `#cad4e0` | Soft shadow |

### Principles
*   **Projection**: Use isometric projection to display 3D objects on a 2D screen without perspective distortion.
*   **Materiality**: Mimic materials like clay, plastic, or glass.
*   **Softness**: Use rounded shapes and soft lighting.

## Usage Example

```css
.iso-card {
  background: white;
  border-radius: 20px;
  box-shadow: 20px 20px 60px #cad4e0, 
              -20px -20px 60px #ffffff;
  transform: perspective(1000px) rotateY(-5deg) rotateX(5deg);
  transition: transform 0.3s ease;
}

.iso-card:hover {
  transform: perspective(1000px) rotateY(0deg) rotateX(0deg) scale(1.02);
}
```
