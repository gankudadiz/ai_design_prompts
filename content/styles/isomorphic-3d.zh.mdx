# 等距 3D (Isomorphic 3D)

等距 3D（通常简称为 3D 插图）为网页设计带来了深度和维度。它通常使用等轴测视角，其中 x、y 和 z 轴相隔 120 度。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-color` | `#f0f2f5` | 柔和背景 |
| `--primary` | `#6c63ff` | 活力强调色 |
| `--shadow` | `#cad4e0` | 柔和阴影 |

### 原则
*   **投影**: 使用等轴测投影在 2D 屏幕上显示 3D 物体，而没有透视失真。
*   **材质**: 模仿粘土、塑料或玻璃等材质。
*   **柔和**: 使用圆形形状和柔和的灯光。

## 使用示例

```css
.iso-card {
  background: white;
  border-radius: 20px;
  box-shadow: 20px 20px 60px #cad4e0, 
              -20px -20px 60px #ffffff;
  transform: perspective(1000px) rotateY(-5deg) rotateX(5deg);
  transition: transform 0.3s ease;
}

.iso-card:hover {
  transform: perspective(1000px) rotateY(0deg) rotateX(0deg) scale(1.02);
}
```
