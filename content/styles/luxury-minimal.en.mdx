# Luxury Minimal

Luxury Minimal combines minimalist design principles with high-end luxury aesthetics. It emphasizes "less is more" through refined details and high-quality material feel.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-void` | `#0a0a0a` | Main background |
| `--gold-champagne` | `#d4af37` | Primary accent |
| `--gold-light` | `#c9a227` | Secondary accent |
| `--charcoal` | `#2a2a2a` | Borders/Separators |
| `--text-white` | `#f5f5f5` | Body text |

### Effects
*   **Gold Gradient**: `linear-gradient(135deg, #d4af37, #c9a227)`
*   **Subtle Accent**: Gold border lines on bottom/left
*   **Text Shadow**: Low opacity gold `text-shadow`
