# 奢华极简 (Luxury Minimal)

奢华极简风格融合了极简主义的设计原则与高端奢侈品的美学。它强调"少即是多"的理念，通过精致的细节和高质量的材质感传达优雅。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-void` | `#0a0a0a` | 主背景 |
| `--gold-champagne` | `#d4af37` | 主要强调色 |
| `--gold-light` | `#c9a227` | 次要强调色 |
| `--charcoal` | `#2a2a2a` | 边框/分隔 |
| `--text-white` | `#f5f5f5` | 正文文本 |

### 效果
*   **金色渐变**: `linear-gradient(135deg, #d4af37, #c9a227)`
*   **微光边框**: 底部/左侧金色边框线
*   **文字阴影**: 低透明度金色 `text-shadow`
