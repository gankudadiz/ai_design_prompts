# Minimalism

Minimalism is a design trend that started in the 20th century and continues today. It is characterized by simplicity, clarity, and "less is more".

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-color` | `#ffffff` | Background |
| `--text-color` | `#000000` | Text/Borders |
| `--accent` | `#cccccc` | Subtle highlights |

### Principles
*   **Whitespace**: The most critical element. Use it to separate groups and focus attention.
*   **Typography**: Use a limited number of typefaces and weights.
*   **Grid**: Stick to a strict grid system for alignment.

## Usage Example

```css
.minimal-container {
  max-width: 800px;
  margin: 0 auto;
  padding: 60px 20px;
  text-align: center;
}

.minimal-btn {
  background: transparent;
  border: 1px solid black;
  color: black;
  padding: 12px 24px;
  text-transform: uppercase;
  letter-spacing: 1px;
  transition: all 0.3s;
}

.minimal-btn:hover {
  background: black;
  color: white;
}
```
