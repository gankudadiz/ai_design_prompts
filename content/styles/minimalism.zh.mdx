# 极简主义 (Minimalism)

极简主义是一种始于 20 世纪并延续至今的设计趋势。其特点是简单、清晰和“少即是多”。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-color` | `#ffffff` | 背景 |
| `--text-color` | `#000000` | 文本/边框 |
| `--accent` | `#cccccc` | 微妙高光 |

### 原则
*   **留白**: 最关键的元素。用它来分隔组并集中注意力。
*   **排版**: 使用数量有限的字体和字重。
*   **网格**: 坚持严格的网格系统进行对齐。

## 使用示例

```css
.minimal-container {
  max-width: 800px;
  margin: 0 auto;
  padding: 60px 20px;
  text-align: center;
}

.minimal-btn {
  background: transparent;
  border: 1px solid black;
  color: black;
  padding: 12px 24px;
  text-transform: uppercase;
  letter-spacing: 1px;
  transition: all 0.3s;
}

.minimal-btn:hover {
  background: black;
  color: white;
}
```
