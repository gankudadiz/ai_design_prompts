# Mint Dreamcore

A style that feels like a lucid dream in a digital space. High contrast between the deep purple background and the piercing mint green creates a striking visual hierarchy.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-dream` | `#0f0a1a` | Main background |
| `--neon-mint` | `#00ffcc` | Primary glow |
| `--neon-pink` | `#ff6b9d` | Secondary glow |
| `--text-lavender` | `#e0d9f5` | Body text |

### CSS Effects
**Mint Glow**:
```css
text-shadow: 0 0 15px #00ffcc;
```

**Pink Glow**:
```css
text-shadow: 0 0 10px #ff6b9d;
```

## Usage Example

```css
.hero-title {
  color: #00ffcc;
  text-shadow: 0 0 15px #00ffcc;
}

.btn:hover {
  background: #00ffcc;
  color: #0f0a1a;
  box-shadow: 0 0 20px #00ffcc;
}
```
