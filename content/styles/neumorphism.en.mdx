# Neumorphism

Neumorphism (New Skeuomorphism) is a visual style that uses highlights and shadows to make elements appear as if they are floating above or pressed into the surface.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-color` | `#e0e5ec` | Main background |
| `--shadow-light` | `#ffffff` | Top-left highlight |
| `--shadow-dark` | `#bebebe` | Bottom-right shadow |
| `--text-color` | `#4a4a4a` | Primary text |

### Effects
*   **Box Shadow (Flat)**: `20px 20px 60px #bebebe, -20px -20px 60px #ffffff`
*   **Box Shadow (Pressed)**: `inset 20px 20px 60px #bebebe, inset -20px -20px 60px #ffffff`
*   **Border Radius**: `50px` (Soft shapes require high border radius)

## Usage Example

```css
.neumorphic-btn {
  border-radius: 50px;
  background: #e0e5ec;
  box-shadow:  20px 20px 60px #bebebe,
               -20px -20px 60px #ffffff;
  border: none;
  color: #4a4a4a;
}

.neumorphic-btn:active {
  box-shadow: inset 20px 20px 60px #bebebe,
              inset -20px -20px 60px #ffffff;
}
```
