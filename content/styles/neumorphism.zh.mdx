# 新拟态 (Neumorphism)

新拟态（新拟物化）是一种视觉风格，利用高光和阴影使元素看起来像是漂浮在表面之上或被压入表面之中。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-color` | `#e0e5ec` | 主要背景 |
| `--shadow-light` | `#ffffff` | 左上角高光 |
| `--shadow-dark` | `#bebebe` | 右下角阴影 |
| `--text-color` | `#4a4a4a` | 主要文本 |

### 效果
*   **盒阴影 (平坦)**: `20px 20px 60px #bebebe, -20px -20px 60px #ffffff`
*   **盒阴影 (按下)**: `inset 20px 20px 60px #bebebe, inset -20px -20px 60px #ffffff`
*   **圆角半径**: `50px` (柔和的形状需要大圆角半径)

## 使用示例

```css
.neumorphic-btn {
  border-radius: 50px;
  background: #e0e5ec;
  box-shadow:  20px 20px 60px #bebebe,
               -20px -20px 60px #ffffff;
  border: none;
  color: #4a4a4a;
}

.neumorphic-btn:active {
  box-shadow: inset 20px 20px 60px #bebebe,
              inset -20px -20px 60px #ffffff;
}
```
