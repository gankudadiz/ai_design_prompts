# Rainbow Y2K

Embrace the chaotic energy of the early 2000s web with a modern twist. This style combines the structure of a terminal with the vibrant colors of vaporwave.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-void` | `#0a0a1a` | Main background |
| `--neon-cyan` | `#00d4ff` | Primary Accent |
| `--neon-magenta` | `#ff00ff` | Secondary Accent |
| `--neon-lime` | `#a3e635` | Highlight Accent |
| `--text-light` | `#e0f0ff` | Body text |

### CSS Effects
**Rainbow Text**:
```css
background: linear-gradient(90deg, #00d4ff, #ff00ff, #a3e635);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

**Neon Glow**:
```css
text-shadow: 0 0 10px #ff00ff;
```

## Usage Example

```css
.hero-title {
  background: linear-gradient(90deg, #00d4ff, #ff00ff, #a3e635);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  animation: rainbow 3s linear infinite;
}
```
