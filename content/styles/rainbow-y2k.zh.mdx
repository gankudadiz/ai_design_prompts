# Rainbow Y2K (千禧彩虹)

拥抱 2000 年代早期网络的混乱能量，并融入现代感。这种风格结合了终端的结构与蒸汽波的鲜艳色彩。

## 设计变量 (Design Tokens)

### 颜色
| Token | 值 | 用途 |
| :--- | :--- | :--- |
| `--bg-void` | `#0a0a1a` | 主背景 |
| `--neon-cyan` | `#00d4ff` | 主强调色 |
| `--neon-magenta` | `#ff00ff` | 次要强调色 |
| `--neon-lime` | `#a3e635` | 高亮色 |
| `--text-light` | `#e0f0ff` | 正文文本 |

### CSS 特效
**彩虹文字**:
```css
background: linear-gradient(90deg, #00d4ff, #ff00ff, #a3e635);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
```

**霓虹辉光**:
```css
text-shadow: 0 0 10px #ff00ff;
```

## 使用示例

```css
.hero-title {
  background: linear-gradient(90deg, #00d4ff, #ff00ff, #a3e635);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  animation: rainbow 3s linear infinite;
}
```
