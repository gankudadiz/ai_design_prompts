# Retro Pixel

Retro Pixel style mimics the graphics of early computer systems and video game consoles (8-bit and 16-bit eras).

## Design Tokens

### Colors (Pico-8 Palette)
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-dark` | `#1d2b53` | Background |
| `--accent-red` | `#ff004d` | Danger/Action |
| `--accent-green` | `#00e436` | Success |
| `--text-light` | `#fff1e8` | Text |

### Effects
*   **Font**: `'Press Start 2P', cursive`
*   **Border**: `4px solid white`
*   **Shadow**: `4px 4px 0px black`

## Usage Example

```css
@import url('https://fonts.googleapis.com/css2?family=Press+Start+2P&display=swap');

.retro-container {
  font-family: 'Press Start 2P', cursive;
  background-color: #1d2b53;
  color: #fff1e8;
  border: 4px solid white;
  box-shadow: 8px 8px 0px black;
  padding: 20px;
  image-rendering: pixelated;
}
```
