# 复古像素 (Retro Pixel)

复古像素风格模仿早期计算机系统和视频游戏机（8 位和 16 位时代）的图形。

## 设计变量

### 颜色 (Pico-8 调色板)
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-dark` | `#1d2b53` | 背景 |
| `--accent-red` | `#ff004d` | 危险/动作 |
| `--accent-green` | `#00e436` | 成功 |
| `--text-light` | `#fff1e8` | 文本 |

### 效果
*   **字体**: `'Press Start 2P', cursive`
*   **边框**: `4px solid white`
*   **阴影**: `4px 4px 0px black`

## 使用示例

```css
@import url('https://fonts.googleapis.com/css2?family=Press+Start+2P&display=swap');

.retro-container {
  font-family: 'Press Start 2P', cursive;
  background-color: #1d2b53;
  color: #fff1e8;
  border: 4px solid white;
  box-shadow: 8px 8px 0px black;
  padding: 20px;
  image-rendering: pixelated;
}
```
