# Retro Terminal

The Retro Terminal style draws inspiration from command-line interfaces of the 1980s and 1990s. It features iconic green glowing text, CRT scanline effects, and classic monochrome aesthetics.

## Design Variables

### Colors
| Variable | Value | Usage |
| :--- | :--- | :--- |
| `--terminal-black` | `#0a0a0a` | Primary background |
| `--terminal-green` | `#33ff00` | Primary text and accent |
| `--terminal-dark` | `#1a1a1a` | Secondary background |
| `--terminal-glow` | `#33ff00` | Glow effects |

### Effects
*   **Glow**: `text-shadow: 0 0 2px #33ff00, 0 0 10px #33ff00`
*   **Scanlines**: `background: linear-gradient(to bottom, transparent 50%, rgba(0,0,0,0.3) 51%)` + `background-size: 100% 4px`
*   **Blinking Cursor**: `animation: blink 1s step-end infinite`
