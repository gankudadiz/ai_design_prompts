# 复古终端 (Retro Terminal)

复古终端风格从 1980 年代和 1990 年代的命令行界面中汲取灵感。它具有标志性的绿色发光文本、CRT 扫描线效果和经典的单色美学。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--terminal-black` | `#0a0a0a` | 主要背景 |
| `--terminal-green` | `#33ff00` | 主要文字和强调色 |
| `--terminal-dark` | `#1a1a1a` | 次要背景 |
| `--terminal-glow` | `#33ff00` | 发光效果 |

### 效果
*   **发光**: `text-shadow: 0 0 2px #33ff00, 0 0 10px #33ff00`
*   **扫描线**: `background: linear-gradient(to bottom, transparent 50%, rgba(0,0,0,0.3) 51%)` + `background-size: 100% 4px`
*   **光标闪烁**: `animation: blink 1s step-end infinite`
