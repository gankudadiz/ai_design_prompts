# Skeuomorphism

Skeuomorphism is a design concept where items resemble their real-world counterparts. In UI design, this means using textures, shadows, and highlights to mimic physical materials like glass, wood, or metal.

## Design Tokens

### Colors & Gradients
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--metal-gradient` | `linear-gradient(to bottom, #f9f9f9 0%, #e0e0e0 100%)` | Surfaces |
| `--glossy-btn` | `linear-gradient(to bottom, #4c9eff 0%, #187bf2 100%)` | Buttons |

### Effects
*   **Emboss**: `box-shadow: inset 0 1px 0 rgba(255,255,255,1), 0 1px 2px rgba(0,0,0,0.3)`
*   **Deboss**: `box-shadow: inset 0 2px 4px rgba(0,0,0,0.4), 0 1px 0 rgba(255,255,255,0.5)`
*   **Text Shadow**: `0 -1px 0 rgba(0,0,0,0.5)` (Etched effect)

## Usage Example

```css
.skeuo-button {
  background: linear-gradient(to bottom, #4c9eff 0%, #187bf2 100%);
  border: 1px solid #105cb6;
  border-radius: 4px;
  box-shadow: inset 0 1px 0 rgba(255,255,255,0.3), 0 1px 2px rgba(0,0,0,0.3);
  color: white;
  text-shadow: 0 -1px 0 rgba(0,0,0,0.3);
}

.skeuo-button:active {
  background: linear-gradient(to bottom, #187bf2 0%, #4c9eff 100%);
  box-shadow: inset 0 2px 4px rgba(0,0,0,0.3);
}
```
