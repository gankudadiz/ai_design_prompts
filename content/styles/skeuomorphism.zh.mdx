# 拟物化 (Skeuomorphism)

拟物化是一种设计概念，其中物品模仿其现实世界的对应物。在 UI 设计中，这意味着使用纹理、阴影和高光来模仿玻璃、木材或金属等物理材料。

## 设计变量

### 颜色 & 渐变
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--metal-gradient` | `linear-gradient(to bottom, #f9f9f9 0%, #e0e0e0 100%)` | 表面 |
| `--glossy-btn` | `linear-gradient(to bottom, #4c9eff 0%, #187bf2 100%)` | 按钮 |

### 效果
*   **浮雕**: `box-shadow: inset 0 1px 0 rgba(255,255,255,1), 0 1px 2px rgba(0,0,0,0.3)`
*   **凹陷**: `box-shadow: inset 0 2px 4px rgba(0,0,0,0.4), 0 1px 0 rgba(255,255,255,0.5)`
*   **文本阴影**: `0 -1px 0 rgba(0,0,0,0.5)` (蚀刻效果)

## 使用示例

```css
.skeuo-button {
  background: linear-gradient(to bottom, #4c9eff 0%, #187bf2 100%);
  border: 1px solid #105cb6;
  border-radius: 4px;
  box-shadow: inset 0 1px 0 rgba(255,255,255,0.3), 0 1px 2px rgba(0,0,0,0.3);
  color: white;
  text-shadow: 0 -1px 0 rgba(0,0,0,0.3);
}

.skeuo-button:active {
  background: linear-gradient(to bottom, #187bf2 0%, #4c9eff 100%);
  box-shadow: inset 0 2px 4px rgba(0,0,0,0.3);
}
```
