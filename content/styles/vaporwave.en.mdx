# Vaporwave

Vaporwave draws inspiration from 80s-90s sci-fi movies, pop culture, and early internet aesthetics. It represents a nostalgic imagination of the future.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-deep` | `#1a1a2e` | Main background |
| `--neon-pink` | `#ff71ce` | Primary accent |
| `--neon-cyan` | `#01cdfe` | Secondary accent |
| `--neon-mint` | `#05ffa1` | Highlight |
| `--neon-purple` | `#b967ff` | Decorative |
| `--text-light` | `#ffeef8` | Body text |

### Effects
*   **Gradient Text**: `background-clip: text` + gradient background
*   **Glow**: `box-shadow` with neon colors
*   **Grid**: `repeating-linear-gradient` for retro grid pattern

## Usage Example

```css
.vapor-card {
  background: linear-gradient(135deg, rgba(255, 113, 206, 0.1), rgba(45, 27, 78, 0.8));
  border: 1px solid rgba(1, 205, 254, 0.4);
  box-shadow: 0 0 20px rgba(255, 113, 206, 0.2);

}
```
