# 蒸汽波 (Vaporwave)

蒸汽波风格从80-90年代的科幻电影、流行文化和早期互联网美学中汲取灵感。它代表了一种对未来的怀旧想象。

## 设计变量

### 颜色
| 变量 | 值 | 用法 |
| :--- | :--- | :--- |
| `--bg-deep` | `#1a1a2e` | 主背景 |
| `--neon-pink` | `#ff71ce` | 主要强调色 |
| `--neon-cyan` | `#01cdfe` | 次要强调色 |
| `--neon-mint` | `#05ffa1` | 高亮色 |
| `--neon-purple` | `#b967ff` | 装饰色 |
| `--text-light` | `#ffeef8` | 正文文本 |

### 效果
*   **渐变文字**: `background-clip: text` + 渐变色背景
*   **发光**: `box-shadow` 配合霓虹色
*   **网格**: `repeating-linear-gradient` 创建复古网格

## 使用示例

```css
.vapor-card {
  background: linear-gradient(135deg, rgba(255, 113, 206, 0.1), rgba(45, 27, 78, 0.8));
  border: 1px solid rgba(1, 205, 254, 0.4);
  box-shadow: 0 0 20px rgba(255, 113, 206, 0.2);
  color: #ffeef8;
  padding: 20px;
}
```
