# Warm Amber

Inspired by the amber monochrome monitors of the 1980s. This style is easier on the eyes than pure high-contrast themes and evokes a strong sense of nostalgia.

## Design Tokens

### Colors
| Token | Value | Usage |
| :--- | :--- | :--- |
| `--bg-warm` | `#1a1200` | Main background |
| `--amber-primary` | `#ffb020` | Primary text/border |
| `--amber-light` | `#ffcc33` | Highlights |
| `--text-beige` | `#e0d5c0` | Body text |
| `--border-warm` | `#2a1e0a` | Borders |

## Usage Example

```css
body {
  background-color: #1a1200;
  color: #e0d5c0;
  font-family: 'Courier New', monospace;
}

a {
  color: #ffb020;
}

a:hover {
  color: #ffcc33;
}
```
