# Warm Amber (暖阳琥珀)

灵感来自 80 年代的琥珀色单色显示器。这种风格比纯高对比度主题更护眼，并唤起强烈的怀旧感。

## 设计变量 (Design Tokens)

### 颜色
| Token | 值 | 用途 |
| :--- | :--- | :--- |
| `--bg-warm` | `#1a1200` | 主背景 |
| `--amber-primary` | `#ffb020` | 主要文本/边框 |
| `--amber-light` | `#ffcc33` | 高亮 |
| `--text-beige` | `#e0d5c0` | 正文文本 |
| `--border-warm` | `#2a1e0a` | 边框 |

## 使用示例

```css
body {
  background-color: #1a1200;
  color: #e0d5c0;
  font-family: 'Courier New', monospace;
}

a {
  color: #ffb020;
}

a:hover {
  color: #ffcc33;
}
```
