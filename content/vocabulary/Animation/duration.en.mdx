# Animation Duration

The **animation-duration** property specifies the length of time that an animation takes to complete one cycle. The default value is `0s`, which means no animation.

## Property Values

*   **Time**: A time value in seconds (`s`) or milliseconds (`ms`). For example, `2s`, `500ms`.

```css
.slow {
  animation-duration: 3s;
}

.fast {
  animation-duration: 0.5s;
}
```

## Notes

*   If negative, it is treated as 0s.
*   This property is required; otherwise, the animation will not play (since duration defaults to 0).
