# Animation Duration (持续时间)

**animation-duration** 属性指定动画完成一个周期所需的时间长度。默认值为 `0s`，表示无动画。

## 属性值

*   **Time**: 以秒 (`s`) 或毫秒 (`ms`) 为单位的时间值。例如 `2s`, `500ms`。

```css
.slow {
  animation-duration: 3s;
}

.fast {
  animation-duration: 0.5s;
}
```

## 注意事项

*   如果是负值，该值会被视为 0s。
*   该属性是必须的，否则动画不会播放（因为持续时间默认为 0）。
