# Easing

The **animation-timing-function** property specifies the speed curve of an animation. It determines whether the animation is constant speed, accelerating, or decelerating during playback.

## Common Values

1.  `linear`: Constant speed.
2.  `ease`: Default. Starts slow, becomes faster, then slows down at the end.
3.  `ease-in`: Starts slow.
4.  `ease-out`: Ends slow.
5.  `ease-in-out`: Starts and ends slow.
6.  `cubic-bezier(n,n,n,n)`: Custom Bezier curve.

```css
.smooth {
  animation-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
}
```
