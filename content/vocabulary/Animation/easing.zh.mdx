# Easing (缓动)

**animation-timing-function** 属性规定动画的速度曲线。它决定了动画在播放期间是匀速、加速还是减速。

## 常用值

1.  `linear`: 匀速。
2.  `ease`: 默认。低速开始，然后加快，在结束前变慢。
3.  `ease-in`: 低速开始。
4.  `ease-out`: 低速结束。
5.  `ease-in-out`: 低速开始和结束。
6.  `cubic-bezier(n,n,n,n)`: 自定义贝塞尔曲线。

```css
.smooth {
  animation-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
}
```
