# Infinite Animation

The **animation-iteration-count** property defines how many times an animation should be played.

## Property Values

*   **Number**: The number of times to play, e.g., `1`, `2.5`.
*   **infinite**: Specifies that the animation should play infinitely.

```css
.loop {
  animation-iteration-count: infinite;
}
```

## Use Cases

Infinite loops are commonly used for:
*   Loading indicators (Spinners)
*   Pulse effects
*   Background scrolling
*   Attention seekers
