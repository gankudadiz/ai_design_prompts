# Infinite Animation (无限循环)

**animation-iteration-count** 属性定义动画应该播放多少次。

## 属性值

*   **Number**: 播放次数，如 `1`, `2.5`。
*   **infinite**: 指定动画应该无限次播放。

```css
.loop {
  animation-iteration-count: infinite;
}
```

## 应用场景

无限循环常用于：
*   加载指示器 (Spinners)
*   脉冲效果 (Pulse effects)
*   背景滚动
*   注意力引导 (Attention seekers)
