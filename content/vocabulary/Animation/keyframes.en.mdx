# Keyframes

The **@keyframes** rule is the core of CSS animations. It allows you to create animations by specifying the state of CSS styles at specific points in the animation sequence.

## Syntax Structure

You can use percentages to specify when changes happen, or use the keywords `from` (equivalent to 0%) and `to` (equivalent to 100%).

```css
@keyframes slideIn {
  from {
    transform: translateX(-100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

@keyframes pulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.1); }
  100% { transform: scale(1); }
}
```

## Core Concepts

1.  **Identifier**: The name of the animation (e.g., `slideIn`, `pulse` above), referenced in `animation-name`.
2.  **Keyframe Selectors**: Percentage values from `0%` to `100%` indicating progress through the animation cycle.
3.  **CSS Properties**: The style properties contained within each keyframe block.
