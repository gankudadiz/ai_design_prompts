# Keyframes (关键帧)

**@keyframes** 规则是 CSS 动画的核心。它允许你通过指定 CSS 样式在动画序列中特定时间点的状态来创建动画。

## 语法结构

你可以使用百分比来规定变化发生的时间，或者使用关键字 `from` (等同于 0%) 和 `to` (等同于 100%)。

```css
@keyframes slideIn {
  from {
    transform: translateX(-100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

@keyframes pulse {
  0% { transform: scale(1); }
  50% { transform: scale(1.1); }
  100% { transform: scale(1); }
}
```

## 核心概念

1.  **标识符 (Identifier)**: 动画的名称（如上例中的 `slideIn`, `pulse`），在 `animation-name` 中引用。
2.  **关键帧选择器**: `0%` 到 `100%` 的百分比值，表示动画周期的进度。
3.  **CSS 属性**: 每个关键帧块中包含的样式属性。
