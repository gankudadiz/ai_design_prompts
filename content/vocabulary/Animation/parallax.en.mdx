# Parallax

**Parallax Scrolling** is a visual effect where the background image moves slower than the foreground content, creating a sense of 3D depth and immersion on a 2D screen.

## Implementation Principles

1.  **CSS**: Using `background-attachment: fixed` allows for the simplest parallax effect.
2.  **JavaScript**: Listening to the `scroll` event and dynamically calculating the `transform: translateY()` value of elements based on scroll distance.
3.  **Libraries**: Using libraries like `Framer Motion`, `GSAP`, or `React Spring` can achieve smoother results.

```css
/* Simple CSS Parallax */
.parallax-bg {
  background-image: url('stars.jpg');
  height: 500px;
  background-attachment: fixed;
  background-position: center;
  background-repeat: no-repeat;
  background-size: cover;
}
```

## Design Considerations

*   **Performance**: Overusing parallax can cause page jank. Always use `transform` instead of `top/left`.
*   **Accessibility**: For users prone to Motion Sickness, provide an option to disable animations (`prefers-reduced-motion`).
