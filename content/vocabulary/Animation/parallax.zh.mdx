# Parallax (视差滚动)

**Parallax Scrolling**（视差滚动）是一种视觉效果，背景图像的移动速度慢于前景内容，从而在 2D 屏幕上营造出 3D 的深度感和沉浸感。

## 实现原理

1.  **CSS**: 使用 `background-attachment: fixed` 可以实现最简单的视差效果。
2.  **JavaScript**: 监听滚动事件 (`scroll`)，根据滚动距离动态计算元素的 `transform: translateY()` 值。
3.  **库**: 使用 `Framer Motion`, `GSAP` 或 `React Spring` 等库可以更平滑地实现。

```css
/* 简易 CSS 视差 */
.parallax-bg {
  background-image: url('stars.jpg');
  height: 500px;
  background-attachment: fixed;
  background-position: center;
  background-repeat: no-repeat;
  background-size: cover;
}
```

## 设计考量

*   **性能**: 滥用视差可能会导致页面卡顿，务必使用 `transform` 而非 `top/left`。
*   **可访问性**: 对于容易产生晕动症（Motion Sickness）的用户，应提供关闭动画的选项（`prefers-reduced-motion`）。
