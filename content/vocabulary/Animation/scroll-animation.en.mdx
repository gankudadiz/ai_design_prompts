> Animations that are triggered or controlled by the scroll position.

## Usage

Scroll animations can add depth and engagement to a website. They are often used for:

- Parallax effects
- Revealing elements as they enter the viewport
- Progress indicators
- Storytelling experiences

## Examples

### Scroll-triggered Fade In

```css
.reveal {
  opacity: 0;
  transform: translateY(20px);
  transition: all 0.6s ease-out;
}

.reveal.active {
  opacity: 1;
  transform: translateY(0);
}
```

```javascript
// JavaScript to toggle class on scroll
window.addEventListener('scroll', reveal);

function reveal() {
  var reveals = document.querySelectorAll('.reveal');
  for (var i = 0; i < reveals.length; i++) {
    var windowHeight = window.innerHeight;
    var elementTop = reveals[i].getBoundingClientRect().top;
    var elementVisible = 150;
    if (elementTop < windowHeight - elementVisible) {
      reveals[i].classList.add('active');
    } else {
      reveals[i].classList.remove('active');
    }
  }
}
```

## Best Practices

- **Performance**: Use efficient properties like `transform` and `opacity` to avoid layout thrashing.
- **Accessibility**: Provide an option to disable animations for users who prefer reduced motion.
- **Subtlety**: Don't overdo it; too much movement can be distracting or cause motion sickness.
