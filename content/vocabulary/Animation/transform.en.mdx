# Transform

The **Transform** property allows you to apply 2D or 3D transformations to elements. While not an animation property itself, it is key to creating high-performance animations (especially movement and scaling) because `transform` does not trigger Reflow, only Repaint or Composite.

## Common Functions

1.  `translate(x, y)`: Moves the element.
2.  `scale(x, y)`: Resizes the element.
3.  `rotate(angle)`: Rotates the element.
4.  `skew(x-angle, y-angle)`: Skews the element.

```css
.card:hover {
  transform: translateY(-5px) scale(1.05);
}
```

## Hardware Acceleration

Using `transform: translate3d(0,0,0)` or `will-change: transform` can enable GPU hardware acceleration, making animations smoother.
