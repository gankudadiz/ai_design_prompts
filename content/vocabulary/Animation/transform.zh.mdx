# Transform (变换)

**Transform** 属性允许你对元素进行 2D 或 3D 转换。虽然它本身不是动画属性，但它是创建高性能动画（尤其是移动和缩放）的关键，因为 transform 不会触发重排（Reflow），只会触发重绘（Repaint）或合成（Composite）。

## 常用函数

1.  `translate(x, y)`: 平移。
2.  `scale(x, y)`: 缩放。
3.  `rotate(angle)`: 旋转。
4.  `skew(x-angle, y-angle)`: 倾斜。

```css
.card:hover {
  transform: translateY(-5px) scale(1.05);
}
```

## 硬件加速

使用 `transform: translate3d(0,0,0)` 或 `will-change: transform` 可以开启 GPU 硬件加速，使动画更流畅。
