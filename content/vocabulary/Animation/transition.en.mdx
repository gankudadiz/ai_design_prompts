# Transition

**Transitions** provide a way to control animation speed when changing CSS properties. Instead of having property changes take effect immediately, you can cause the changes in a property to take place over a period of time.

## Core Properties

1.  `transition-property`: What to animate (e.g., `opacity`, `transform`, `all`).
2.  `transition-duration`: How long it takes (e.g., `0.3s`, `300ms`).
3.  `transition-timing-function`: The speed curve (e.g., `ease`, `linear`, `cubic-bezier`).
4.  `transition-delay`: Time to wait before starting.

## Shorthand

```css
/* property duration timing-function delay */
.box {
  transition: all 0.3s ease-in-out;
}
```
