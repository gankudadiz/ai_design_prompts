# Transition (过渡)

**Transitions** (过渡) 提供了一种控制 CSS 属性变化时动画速度的方法。你可以让属性的变化在一段时间内发生，而不是立即生效。

## 核心属性

1.  `transition-property`: 要动画化的属性 (例如 `opacity`, `transform`, `all`)。
2.  `transition-duration`: 持续时间 (例如 `0.3s`, `300ms`)。
3.  `transition-timing-function`: 速度曲线 (例如 `ease`, `linear`, `cubic-bezier`)。
4.  `transition-delay`: 开始前的延迟时间。

## 简写

```css
/* 属性 持续时间 速度曲线 延迟 */
.box {
  transition: all 0.3s ease-in-out;
}
```
