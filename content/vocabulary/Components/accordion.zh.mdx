# Accordion (手风琴)

手风琴允许用户切换内容部分的显示。

## 用法

*   组织大量内容。
*   常见问题解答 (FAQ) 部分。

```html
<details>
  <summary>第一部分</summary>
  <p>第一部分的内容。</p>
</details>
```
