> An **Alert** is a component that displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

## Types (Severity)
-   **Success**: Indicates a successful or positive action (usually green).
-   **Info**: Provides neutral information (usually blue).
-   **Warning**: Warns of potential issues (usually yellow/orange).
-   **Error**: Indicates a critical issue or failure (usually red).

## Anatomy
-   **Icon**: Visual indicator of severity.
-   **Title**: Brief summary.
-   **Description**: Detailed message.
-   **Close Button**: Optional button to dismiss the alert.

## Usage
-   **Form Feedback**: "Your changes have been saved."
-   **System Status**: "System maintenance scheduled for tonight."
-   **Validation Errors**: "Please correct the errors below."
