# Avatar

Avatars are used to represent a user or an entity (like a team).

## Usage

*   Display user profile pictures.
*   Show initials if no image is available.

```css
.avatar {
  vertical-align: middle;
  width: 50px;
  height: 50px;
  border-radius: 50%;
}
```
