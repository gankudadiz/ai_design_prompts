# Avatar (头像)

头像用于代表用户或实体（如团队）。

## 用法

*   显示用户个人资料图片。
*   如果不可用图像，显示首字母。

```css
.avatar {
  vertical-align: middle;
  width: 50px;
  height: 50px;
  border-radius: 50%;
}
```
