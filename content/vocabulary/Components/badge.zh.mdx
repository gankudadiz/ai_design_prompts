# Badge (徽章)

徽章用于突出显示项目的状态，以便快速识别。

## 用法

*   显示通知计数（例如，未读消息）。
*   指示状态（例如，“新”，“待处理”）。

```html
<span class="badge">New</span>
<span class="badge badge-primary">4</span>
```
