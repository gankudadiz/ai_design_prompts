# Breadcrumb (面包屑)

面包屑允许用户跟踪他们在程序、文档或网站中的位置。

## 用法

*   显示层次结构（例如，首页 > 产品 > 电子产品）。
*   允许快速导航到父类别。

```html
<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#">首页</a></li>
    <li class="breadcrumb-item active" aria-current="page">库</li>
  </ol>
</nav>
```
