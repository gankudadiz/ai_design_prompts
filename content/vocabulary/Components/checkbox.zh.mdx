# Checkbox (复选框)

复选框允许用户从一组选项中选择一个或多个项目。

## 用法

*   当用户可以选择多个选项时使用。
*   用于二进制的是/否选择（例如，“我同意条款”）。

```html
<input type="checkbox" id="subscribe" name="subscribe" />
<label for="subscribe">订阅时事通讯</label>
```
