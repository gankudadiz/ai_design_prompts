# Input

Inputs allow users to enter text into a UI. They typically appear in forms and dialogs.

## Types

1.  **Text**: Standard single-line text.
2.  **Password**: Masked characters for security.
3.  **Email**: Optimized for email addresses.
4.  **Number**: Restricted to numeric input.

```html
<input type="text" placeholder="Enter your name" />
```
