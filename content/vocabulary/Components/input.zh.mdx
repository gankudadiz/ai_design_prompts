# Input (输入框)

输入框允许用户在 UI 中输入文本。它们通常出现在表单和对话框中。

## 类型

1.  **文本 (Text)**: 标准单行文本。
2.  **密码 (Password)**: 用于安全目的的掩码字符。
3.  **邮箱 (Email)**: 针对电子邮件地址进行了优化。
4.  **数字 (Number)**: 仅限数字输入。

```html
<input type="text" placeholder="输入你的名字" />
```
