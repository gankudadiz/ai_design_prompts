# Modal

Modals disable the main content and require the user to interact with the dialog before returning.

## Usage

*   Use for critical confirmations (e.g., "Delete this item?").
*   Use for complex forms that need focus.

```css
/* Overlay style */
.modal-overlay {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: rgba(0, 0, 0, 0.5);
}
```
