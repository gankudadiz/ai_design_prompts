# Modal (模态框)

模态框会禁用主要内容，并要求用户在返回之前与对话框进行交互。

## 用法

*   用于关键确认（例如，“删除此项目？”）。
*   用于需要聚焦的复杂表单。

```css
/* 遮罩层样式 */
.modal-overlay {
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: rgba(0, 0, 0, 0.5);
}
```
