# Pagination (分页)

分页允许用户浏览大量内容。

## 用法

*   搜索结果列表。
*   包含许多行的表格数据。

```html
<div class="pagination">
  <a href="#">&laquo;</a>
  <a href="#">1</a>
  <a href="#" class="active">2</a>
  <a href="#">3</a>
  <a href="#">&raquo;</a>
</div>
```
