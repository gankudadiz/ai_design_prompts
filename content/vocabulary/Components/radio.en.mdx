# Radio Button

Radio buttons allow the user to select exactly one option from a set.

## Usage

*   Use when users must choose only one option.
*   All options should be visible side-by-side.

```html
<input type="radio" id="option1" name="group" value="1" />
<label for="option1">Option 1</label>
<input type="radio" id="option2" name="group" value="2" />
<label for="option2">Option 2</label>
```
