# Radio Button (单选按钮)

单选按钮允许用户从一组选项中恰好选择一个选项。

## 用法

*   当用户必须只选择一个选项时使用。
*   所有选项应并排可见。

```html
<input type="radio" id="option1" name="group" value="1" />
<label for="option1">选项 1</label>
<input type="radio" id="option2" name="group" value="2" />
<label for="option2">选项 2</label>
```
