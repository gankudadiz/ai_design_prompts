# Slider (滑块)

滑块反映了条形图上的一系列值，用户可以从中选择单个值。

## 用法

*   调整音量或亮度等设置。
*   选择价格范围。

```html
<input type="range" min="1" max="100" value="50" class="slider" id="myRange">
```
