# Spinner

Spinners are used for indeterminate wait times.

## Usage

*   Use when the wait time is unknown.
*   Place inside buttons or center on screen.

```css
.loader {
  border: 16px solid #f3f3f3;
  border-top: 16px solid #3498db;
  border-radius: 50%;
  width: 120px;
  height: 120px;
  animation: spin 2s linear infinite;
}
```
