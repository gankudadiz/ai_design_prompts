# Spinner (加载器)

加载器用于不确定的等待时间。

## 用法

*   当等待时间未知时使用。
*   放置在按钮内部或屏幕中央。

```css
.loader {
  border: 16px solid #f3f3f3;
  border-top: 16px solid #3498db;
  border-radius: 50%;
  width: 120px;
  height: 120px;
  animation: spin 2s linear infinite;
}
```
