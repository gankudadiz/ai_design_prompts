# Switch

Switches toggle the state of a single setting on or off.

## Usage

*   Use for instant actions that don't require a save button.
*   Alternative to a checkbox for "on/off" states.

```css
/* Simple CSS Switch Concept */
.switch {
  position: relative;
  display: inline-block;
  width: 60px;
  height: 34px;
}
```
