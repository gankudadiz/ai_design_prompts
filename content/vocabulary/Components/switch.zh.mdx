# Switch (开关)

开关用于切换单个设置的开启或关闭状态。

## 用法

*   用于不需要保存按钮的即时操作。
*   作为“开启/关闭”状态下复选框的替代方案。

```css
/* 简单的 CSS 开关概念 */
.switch {
  position: relative;
  display: inline-block;
  width: 60px;
  height: 34px;
}
```
