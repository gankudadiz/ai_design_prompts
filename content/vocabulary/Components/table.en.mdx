> A **Table** is a component for displaying data in rows and columns, making it easy to scan, compare, and analyze information.

## Anatomy
-   **Header**: Labels for columns.
-   **Body**: The actual data rows.
-   **Footer**: Summary or pagination (optional).
-   **Pagination**: Navigation for large datasets.

## Features
-   **Sorting**: Clicking headers to sort by column.
-   **Filtering**: Searching or selecting criteria to narrow down results.
-   **Selection**: Checkboxes to select rows for bulk actions.
-   **Striped Rows**: Alternating background colors for readability.

## Best Practices
-   **Alignment**: Align text left, numbers right.
-   **Responsiveness**: Tables are hard on mobile. Consider horizontal scrolling or converting rows to cards (stacked view).
