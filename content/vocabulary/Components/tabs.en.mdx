# Tabs

Tabs organize content into separate views where only one view can be visible at a time.

## Usage

*   Group related content (e.g., "Account", "Settings", "Profile").
*   Switch between different aspects of a dataset.

```html
<div class="tab">
  <button class="tablinks">London</button>
  <button class="tablinks">Paris</button>
  <button class="tablinks">Tokyo</button>
</div>
```
