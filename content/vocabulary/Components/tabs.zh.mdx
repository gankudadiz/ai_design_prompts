# Tabs (选项卡)

选项卡将内容组织成单独的视图，一次只能看到一个视图。

## 用法

*   对相关内容进行分组（例如，“帐户”，“设置”，“个人资料”）。
*   在数据集的不同方面之间切换。

```html
<div class="tab">
  <button class="tablinks">伦敦</button>
  <button class="tablinks">巴黎</button>
  <button class="tablinks">东京</button>
</div>
```
