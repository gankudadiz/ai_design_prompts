> A **Tag** (or Chip/Badge) is a compact element that represents an input, attribute, or action.

## Usage
-   **Categorization**: Labeling items with categories (e.g., "New", "Sale").
-   **Status**: Showing the state of an item (e.g., "Pending", "Approved").
-   **Input**: Entering multiple values (e.g., email recipients).
-   **Filtering**: Showing active filters.

## Types
-   **Static**: Informational only.
-   **Dismissible**: Can be removed (has an 'x' icon).
-   **Clickable**: Acts as a button or link.

## Best Practices
-   **Colors**: Use colors to indicate status or grouping, but ensure text contrast.
-   **Concise**: Keep text short.
