# Textarea (多行文本框)

多行文本框允许用户输入大量自由格式的文本。

## 用法

*   用于评论、反馈表或长篇回答。
*   可以调整大小。

```html
<textarea rows="4" cols="50">
在此输入文本...
</textarea>
```
