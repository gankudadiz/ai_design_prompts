# Toast

Toasts provide brief feedback about an operation through a message at the bottom of the screen.

## Usage

*   Show success or error messages (e.g., "File saved").
*   Disappear automatically after a few seconds.

```javascript
// Concept
showToast("Saved successfully!");
```
