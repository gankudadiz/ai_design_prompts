# Toast (吐司提示)

吐司提示通过屏幕底部的消息提供有关操作的简短反馈。

## 用法

*   显示成功或错误消息（例如，“文件已保存”）。
*   几秒钟后自动消失。

```javascript
// 概念
showToast("保存成功！");
```
