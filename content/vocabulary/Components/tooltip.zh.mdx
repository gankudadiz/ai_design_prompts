# Tooltip (工具提示)

当用户悬停、聚焦或轻触元素时，工具提示会显示信息性文本。

## 用法

*   为图标或按钮提供额外的上下文。
*   保持文本简短扼要。

```html
<button title="这是一个工具提示">悬停我</button>
```
