# 像相框一样的盒子

你可以把网页上的每一个元素都想象成挂在墙上的**相框**。

1.  **内容 (Content)**：照片本身。
2.  **内边距 (Padding)**：照片和相框内边缘之间的留白（卡纸）。
3.  **边框 (Border)**：相框的实体边框。
4.  **外边距 (Margin)**：相框和其他相框之间的距离。

## 为什么我的布局总是“撑破”？

新手最头疼的问题是：*“我明明设置了宽度 100px，加了边框后它却变大了！”*

这是因为默认情况下，CSS 使用的是**标准盒模型**：
> 宽度 = 内容宽度

这就像你买相框时，只说了照片的尺寸，结果加上边框后，整个相框比你预想的要大，挂不进原本预留的墙面空间。

### 现代解决方案：border-box

现代开发通常使用 `box-sizing: border-box`。这就像是你直接告诉店家：“我要一个总宽度 100px 的相框”，不管里面的边框和留白有多厚，总宽度都被锁定为 100px。

```css
/* 推荐设置 */
* {
  box-sizing: border-box;
}
```
