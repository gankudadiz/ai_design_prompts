# 图片变形了怎么办？

在响应式布局中，我们经常给图片设置固定的宽高（例如卡片封面），但上传的图片比例各不相同。直接强制设置宽高会导致图片**拉伸变形**。`object-fit` 就是为了解决这个问题而生的。

## 核心属性值

### 1. fill (默认值)
**“强行填满”**。
- 图片会被拉伸以填满整个容器。
- **后果**：图片比例失调，人变胖或变瘦。

### 2. contain
**“完整显示”**。
- 保持图片原有比例，缩放直到图片能完全放入容器。
- **后果**：容器内可能会有留白（黑边）。

### 3. cover (最常用)
**“裁剪填充”**。
- 保持图片原有比例，缩放直到图片填满整个容器。
- **后果**：图片的一部分会被裁剪掉（通常是边缘），但不会变形且没有留白。
- **场景**：全屏背景图、头像、卡片封面。

```css
.card-image {
  width: 100%;
  height: 200px;
  object-fit: cover; /* 完美适配 */
}
```

### 4. none
**“保持原样”**。
- 图片保持原始尺寸，不做缩放，只显示中间部分（裁剪）。

## 配合 Object-position

如果使用 `object-fit: cover` 裁剪了图片，默认是裁剪掉边缘，保留中心。你可以用 `object-position` 改变关注点。

```css
img {
  object-fit: cover;
  object-position: top center; /* 关注图片的顶部，裁剪底部 */
}
```
