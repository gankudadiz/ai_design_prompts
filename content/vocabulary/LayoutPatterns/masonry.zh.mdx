# 什么是瀑布流？

你一定用过 Pinterest 或者小红书。你会发现它们的图片排版不像普通的网格那样整整齐齐（每行高度一样），而是像**砌墙**一样，长长短短地交错在一起。

这就是**Masonry (砖石/瀑布流)** 布局。它能最大化地利用屏幕空间，不留难看的空白缝隙。

## 怎么实现它？

虽然看起来只是不规则的网格，但实现起来其实挺麻烦的。

### 1. 简单做法 (CSS Columns)
就像报纸排版一样，内容先填满第一列，再填第二列。
*   **缺点**：读起来是从上往下，而不是从左往右。

### 2. 完美做法 (JavaScript)
最常用的方法是用 JavaScript 计算。
1.  准备几列（比如3列）。
2.  拿一个新的方块。
3.  看哪一列目前**最短**。
4.  把方块插到那一列下面。
5.  重复。
