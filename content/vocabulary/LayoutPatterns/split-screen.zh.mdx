# 什么是分屏布局？

分屏布局（Split Screen）将视口均分为两个部分（通常是 50/50）。
这是一种非常现代且极具视觉冲击力的设计，常用于：
*   **登录/注册页**：一边是品牌图，一边是表单。
*   **产品着陆页**：一边是产品特写，一边是介绍文案。

## 代码实现

Flexbox 是最佳选择：

```css
.container {
  display: flex;
  min-height: 100vh;
}

.left, .right {
  flex: 1; /* 均分空间 */
  display: flex;
  justify-content: center; /* 内容居中 */
  align-items: center;
}

/* 移动端适配：改为上下排列 */
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}
```
