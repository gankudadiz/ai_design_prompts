> **`aspect-ratio`** CSS 属性设置盒子的首选纵横比，该比例将用于计算自动尺寸和其他一些布局函数。

## 语法
```css
/* 宽度 / 高度 */
aspect-ratio: 16 / 9;
aspect-ratio: 1 / 1;
aspect-ratio: 0.5;
```

## 主要特点
-   **响应式**：帮助在视口调整大小时保持元素比例（如图像或视频容器）。
-   **简化**：取代了用于创建响应式正方形或矩形的旧“padding-top hack”。

## 用例
-   响应式视频嵌入（例如 16:9）。
-   具有一致图像尺寸的产品卡片。
-   头像占位符。
