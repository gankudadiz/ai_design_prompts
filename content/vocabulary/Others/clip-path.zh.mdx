> **`clip-path`** CSS 属性创建一个裁剪区域，用于设置元素应显示的区域。区域内的部分显示，区域外的部分隐藏。

## 语法
```css
clip-path: circle(50%);
clip-path: polygon(50% 0%, 0% 100%, 100% 100%);
clip-path: path('M0.5,1 C0.5,1,0,0.7,0,0.3 A0.25,0.25,1,1,1,0.5,0.3 A0.25,0.25,1,1,1,1,0.3 C1,0.7,0.5,1,0.5,1 Z');
```

## 主要形状
-   **基本形状**：`circle()`（圆）、`ellipse()`（椭圆）、`polygon()`（多边形）、`inset()`（矩形内嵌）。
-   **SVG 路径**：`path()` 用于复杂的自定义形状。
-   **几何盒子**：使用元素的盒模型（例如 `padding-box`）。

## 用例
-   创建非矩形图像（六边形、星形）。
-   创意揭示动画。
-   在不编辑图像文件的情况下视觉上去除背景。
