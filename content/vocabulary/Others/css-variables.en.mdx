> **CSS Variables** (Custom Properties) are entities defined by CSS authors that contain specific values to be reused throughout a document.

## Usage
- **Declaration**: Defined using the `--*` notation (e.g., `--main-color: black;`).
- **Access**: Accessed using the `var()` function (e.g., `color: var(--main-color);`).

## Key Benefits
1.  **Reusability**: Define a value once and reuse it in multiple places.
2.  **Maintainability**: Update a value in one place to update it globally.
3.  **Dynamic**: Can be updated with JavaScript or changed based on media queries.

## Common Use Cases
-   Color themes (Light/Dark mode).
-   Consistent spacing and sizing units.
-   Storing calculation results.
