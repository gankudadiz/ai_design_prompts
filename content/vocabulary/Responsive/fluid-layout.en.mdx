# Fluid Layout

**Fluid Layout** (also known as Liquid Layout) is a layout technique where element dimensions are defined using relative units (such as percentages `%`, viewport units `vw/vh`, or `fr`) instead of fixed pixel units (`px`). This allows the page to fill any container size, behaving like water.

## Core Concepts

### 1. Relative vs. Fixed Units

*   **Fixed**: `width: 300px` - It always occupies 300px, regardless of screen width. This causes scrollbars on small screens and excessive whitespace on large screens.
*   **Fluid**: `width: 50%` - Always occupies half the width of its parent container.

### 2. Max and Min Width

To prevent fluid elements from becoming too wide (hard to read on large screens) or too narrow (content overflow), `max-width` and `min-width` are often used in conjunction.

```css
.container {
  width: 90%;        /* Fluid: takes up 90% width */
  max-width: 1200px; /* Limit: max 1200px */
  margin: 0 auto;    /* Center */
}
```

## Modern Fluid Layout Technologies

### Flexbox
Flexbox is inherently fluid. The `flex-grow` and `flex-shrink` properties control how elements fill space.

```css
.item {
  flex: 1; /* Automatically divides remaining space */
}
```

### CSS Grid
Grid provides the powerful `fr` (fraction) unit.

```css
.grid {
  display: grid;
  grid-template-columns: 1fr 2fr; /* 1st column takes 1/3, 2nd takes 2/3 */
}
```
