# 告别断点跳跃

在传统的响应式设计中，我们通常这样设置字体：
- 手机：16px
- 平板：18px
- 桌面：24px

这种方式会导致在断点切换瞬间，字体大小发生“跳跃”。**流体排版 (Fluid Typography)** 则让字体大小像水流一样，随着屏幕宽度的变化而平滑缩放。

## 核心工具：`clamp()`

`clamp()` 函数接受三个参数：最小值、首选值、最大值。它是实现流体排版的终极武器。

```css
h1 {
  /* 
    最小值: 2rem (32px)
    首选值: 5vw (视口宽度的 5%)
    最大值: 4rem (64px)
  */
  font-size: clamp(2rem, 5vw, 4rem);
}
```

- **当屏幕很窄时**：字体不会小于 `2rem`，保证可读性。
- **当屏幕很宽时**：字体不会大于 `4rem`，防止标题过大。
- **在中间时**：字体大小等于 `5vw`，随屏幕宽度线性增长。

## 高级技巧：精确控制

如果你希望在特定的屏幕宽度范围内（例如 320px 到 1200px），字体从 16px 精确线性增加到 24px，可以使用 CSS `calc()` 混合单位：

```css
/* 这种复杂的计算通常通过 SCSS 函数或在线生成器完成 */
body {
  font-size: clamp(1rem, 0.8rem + 1vw, 1.5rem);
}
```

流体排版不仅适用于 `font-size`，也同样适用于 `padding`、`margin` 和 `gap`，让整个布局的呼吸感随屏幕尺寸自然伸缩。
