> **边框** 是围绕元素内容和内边距的线条。

## 语法
```css
/* 简写：宽度 样式 颜色 */
border: 1px solid black;

/* 单独属性 */
border-width: 2px;
border-style: dashed;
border-color: red;
```

## 常见样式
-   `solid`: 实线。
-   `dashed`: 虚线（短方形线段）。
-   `dotted`: 点线（圆点）。
-   `double`: 双实线。
-   `none`: 无边框。

## 用例
-   定义卡片或按钮的边缘。
-   在元素之间创建分隔符。
-   高亮显示活动或获得焦点的元素。
