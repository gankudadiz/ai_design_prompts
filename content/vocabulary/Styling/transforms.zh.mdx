> **`transform`** CSS 属性允许你旋转、缩放、倾斜或平移元素。它修改 CSS 视觉格式化模型的坐标空间。

## 语法
```css
transform: translate(10px, 20px);
transform: scale(1.5);
transform: rotate(45deg);
transform: skew(10deg);
transform: translate(10px, 20px) rotate(45deg); /* 多个变换 */
```

## 函数
-   `translate(x, y)`: 移动元素。
-   `scale(x, y)`: 调整元素大小。
-   `rotate(angle)`: 旋转元素（2D 或 3D）。
-   `skew(x-angle, y-angle)`: 扭曲元素。

## 性能
变换通常是硬件加速的（在 GPU 上运行），这使它们成为动画的理想选择，因为它们不会触发昂贵的布局重新计算（重排）。

## 用例
-   居中元素 (`translate(-50%, -50%)`)。
-   悬停时的缩放效果。
-   创建 3D 卡片翻转。
