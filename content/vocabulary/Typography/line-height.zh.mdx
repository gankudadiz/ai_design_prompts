# line-height (行高)

`line-height` 属性用于设置行间的距离。适当的行高对于提升长段文本的可读性至关重要。

## 核心概念

### 1. 无单位数值 (推荐)
使用无单位的数字（如 `1.5`）是最佳实践。它是当前字体大小的倍数。
例如，`font-size: 16px` 且 `line-height: 1.5`，则行高为 `24px`。
**优势**: 子元素继承的是这个比例系数，而不是计算后的绝对像素值。

### 2. 绝对单位 (px, rem)
设置固定的行高。
**缺点**: 如果子元素改变了字体大小，固定的行高可能会导致行间距过窄或重叠。

### 3. 百分比 (%)
类似于无单位数值，但继承时会有差异（通常继承计算后的值）。

## CSS 实现

```css
.text-element {
  /* 最佳实践: 无单位数值 */
  line-height: 1.5;   /* 正文标准，清晰易读 */
  line-height: 1.2;   /* 标题通常更紧凑 */
  line-height: 1.6;   /* 宽松的正文 */

  /* 不推荐 */
  line-height: 24px;
}
```

## 设计建议
*   **正文 (Body)**: 通常设置为 `1.4` 到 `1.6` 之间，以确保阅读舒适。
*   **标题 (Headings)**: 字号较大时，行高应相对减小，通常在 `1.1` 到 `1.3` 之间。
