# Button

Buttons are the most common way users interact with a web application. They communicate actions that users can take.

## Types of Buttons

1.  **Primary**: The main action on a screen (e.g., "Submit", "Save").
2.  **Secondary**: Alternative actions (e.g., "Cancel", "Back").
3.  **Tertiary/Ghost**: Less important actions, often transparent with a border or just text.

## States

A button isn't just a static rectangle. It has multiple states:

*   **Default**: The normal state.
*   **Hover**: When the mouse is over it.
*   **Active/Pressed**: When clicked.
*   **Disabled**: When the action is unavailable.
*   **Focus**: When selected via keyboard.

```css
.btn {
  padding: 10px 20px;
  border-radius: 4px;
  cursor: pointer;
  transition: all 0.2s;
}

.btn-primary {
  background-color: #0070f3;
  color: white;
}

.btn-primary:hover {
  background-color: #0051a2;
}
```
