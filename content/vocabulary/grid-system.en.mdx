# Grid System

The **Grid System** is the invisible structure that holds a design together. Think of it as the skeleton of your interface.

## Why use a Grid?

1.  **Consistency**: Helps align elements across different pages.
2.  **Efficiency**: Speeds up the design-to-code process.
3.  **Responsiveness**: Makes it easier to adapt designs to different screen sizes.

## The 12-Column Grid

The most popular grid system in web design is the **12-column grid**. Why 12? Because 12 is divisible by 1, 2, 3, 4, and 6, making it extremely flexible.

```css
.container {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 20px;
}

.col-4 {
  grid-column: span 4; /* Takes up 4 columns (1/3 of width) */
}

.col-6 {
  grid-column: span 6; /* Takes up 6 columns (1/2 of width) */
}
```
