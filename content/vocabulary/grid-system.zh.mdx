# Grid System (栅格系统)

**Grid System** (栅格系统) 是支撑设计的无形结构。把它想象成界面的骨架。

## 为什么要使用 Grid?

1.  **一致性**: 帮助对齐不同页面上的元素。
2.  **效率**: 加快从设计到代码的过程。
3.  **响应式**: 更容易使设计适应不同的屏幕尺寸。

## 12列栅格

网页设计中最流行的栅格系统是 **12列栅格**。为什么是12？因为12可以被1、2、3、4和6整除，非常灵活。

```css
.container {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 20px;
}

.col-4 {
  grid-column: span 4; /* 占据4列 (宽度的1/3) */
}

.col-6 {
  grid-column: span 6; /* 占据6列 (宽度的1/2) */
}
```
