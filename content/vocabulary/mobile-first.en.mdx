# Mobile First

**Mobile First** is a design philosophy that aims to create better user experiences by starting the design process from the smallest screen: mobile devices.

## Why Mobile First?

1.  **Content Focus**: Forces you to prioritize the most important content.
2.  **Performance**: Mobile code is usually lighter and faster.
3.  **Trend**: More people access the web via mobile than desktop.

## CSS Implementation

In CSS, this means using `min-width` media queries. The default styles are for mobile, and you add complexity as the screen gets wider.

```css
/* Default styles (Mobile) */
.container {
  padding: 10px;
  font-size: 14px;
}

/* Tablet (md) */
@media (min-width: 768px) {
  .container {
    padding: 20px;
    font-size: 16px;
  }
}

/* Desktop (lg) */
@media (min-width: 1024px) {
  .container {
    padding: 40px;
    font-size: 18px;
  }
}
```
