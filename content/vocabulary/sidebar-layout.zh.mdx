# 什么是侧边栏布局？

这是目前 Web 应用（特别是 Dashboard 后台系统）最主流的布局方式。
通常左侧是一个固定的导航菜单，右侧是主要的操作区域。

## 关键特性

1.  **独立滚动**：通常希望侧边栏和主内容能独立滚动。
2.  **响应式**：在手机上，侧边栏通常会隐藏，变成一个汉堡菜单（Hamburger Menu）。

## 代码实现

使用 Grid 实现非常方便：

```css
.container {
  display: grid;
  grid-template-columns: 250px 1fr; /* 左侧固定 250px，右侧自适应 */
  min-height: 100vh;
}

/* 移动端适配 */
@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr; /* 变成单列 */
  }
  aside {
    display: none; /* 或者变成抽屉式菜单 */
  }
}
```
